Run the application to view the main activity screen that contains the list of places retrieved from the sqlite database
click on a place to view the description of the place. you will be directed to the second screen
click the 'back' button to go back to the previous screen
click the '-' button on the second screen to delete the place
To add a new place press the "+" button on the second screen. Empty values get populated on the fields. Enter the details of place you want to add and click "SAVE" button. enter all the values for the place to be added
you will be directed to the main activity where you can view the place added
